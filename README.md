# Bountees Kitchen Inventory Management Subsystem
 The Bountees Kitchen Inventory Management Subsystem is a core module designed to manage and monitor all kitchen-related ingredients and supplies. It provides real-time tracking of stock levels, records inventory movement, and supports timely restocking to prevent shortages. By maintaining accurate and organized inventory data, the subsystem helps streamline kitchen operations, reduce waste, and ensure the continuous availability of essential items needed for food preparation.


# Features of the Subsystem

This subsystem consists of the following functionalities:

* Add Inventory
Allows staff to record newly arrived stock items and store them in the inventory database.

* Inventory Report
Generates a summary of the current inventory, including stock levels, item movements, and low-stock indicators.

* Update Inventory
Enables staff to modify existing item details such as name, category, or description.

* Update Quantity
Records changes in stock quantity due to usage, replenishment, or adjustments.

# Use Case Description

The following use cases describe how the subsystem operates based on the workflow:

* Add Inventory Item
Triggered when new stock arrives. The owner or staff adds the item to the inventory, and it is recorded in the inventory database.

* Update Inventory Item
Triggered when changes occur in item details. Staff update the information, and the system reflects these changes in the database.

* Update Stock Quantity
Triggered when stock is used or replenished. Staff adjust the quantity, and the updated level is saved in the database.

* View Inventory
Triggered when the owner or staff requests to review the inventory. The system displays the current stock levels for reference and planning.

* Generate Inventory Report
Triggered when the owner or staff initiates a report request. The system generates a complete inventory summary for review.

* Notify Low Stock
Triggered when an item falls below the defined threshold. The system generates a low-stock alert for the owner and cashier to take necessary action.
